int hwapi_periodic_add ( void(*)(void)  callback,
int  period

Adds the function callback to the kernel periodic callback functions. If hwapi_periodic_add() successes, the function callback will be called every period timeslots with Kernel Priority.

Parameters
callbackPointer to the periodic function
periodPositive integer, in time slots
Returns
zero on success, -1 on error
