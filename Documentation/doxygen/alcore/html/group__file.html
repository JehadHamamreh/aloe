int hwapi_file_open ( string  name)

Given a pathname for a file, hwapi_file_open() returns a file descriptor, a nonnegative integer for use in hwapi_file_write(), hwapi_file_read() and hwapi_file_close(). The function access the file-system directory configured by a previous call to hwapi_file_set_path().

Parameters
nameName of the file
Returns
nonnegative integer on success, -1 on error.
