int rtdal_periodic_add ( void(*)(void)  callback,
int  period 
)

Adds the function callback to the kernel periodic callback functions. If rtdal_periodic_add() successes, the function callback will be called every period timeslots with Kernel Priority.

Parameters

periodPositive integer, in time slots
