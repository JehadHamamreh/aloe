rtdal_process_t *  process

Adds a process to the pipeline. It is inserted in the position min(n,exec_position) where n is the number of elements in the spscq and exec_position is defined in the process attributes used in the call to rtdal_process_new(). Returns the position it has finally been inserted.

Parameters
objPointer to the pipeline_t object where the process is inserted process pointer to the rtdal_process_t object to insert.
Returns
non-negative integer number indicating the position it has been inserted, or -1 on error.
