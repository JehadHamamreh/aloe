Detailed Description

This module writes the received samples to a file.

Function Documentation

int initialize ( )

Waits to receive buffer_sz samples and then writes the signal to file_name. When buffer_sz new samples are received, the signal is overwritten. 

 If buffer_sz is not provided, the signal is saved each timeslot.
Parameters
data_type0 for float, 1 for _Complex float and 2 for _Complex short
