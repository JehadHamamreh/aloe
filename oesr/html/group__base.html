int oesr_exit ( void *  context)

Stops the calling module execution. Sets the module as NON-runnable.

The module continues the execution after this function returns but, once the module returns from one of the Init/Run/Stop functions, it will not be executed again. The oesr_exit() function DOES NOT free the module's resources. In the case of a normal STOP, this is done by the the OESR using the rtdal_process_remove() function.
