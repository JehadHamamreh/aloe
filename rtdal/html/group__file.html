int rtdal_file_open ( string  name)

Given a pathname for a file, rtdal_file_open() returns a file descriptor, a nonnegative integer for use in rtdal_file_write(), rtdal_file_read() and rtdal_file_close(). The function access the file-system directory configured by a previous call to rtdal_file_set_path().

Parameters
nameName of the file
Returns
nonnegative integer on success, -1 on error.
