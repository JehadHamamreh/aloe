int man_platform_connect ( )

1) For each node, call node[i].ctrlItf.connect() and node[i].probeItf.connect() 3) If syncLocal==false, syncItf.connect() 4) For each node, send Cmd.CONNECT command

int man_platform_disconnect ( )

1) For each node, send Cmd.DISCONNECT command 2) For each node, call node.ctrlItf.close(), node.probeItf.close() 3) call syncItf.close()

int man_platform_update ( )

For each node, send a HWINFO command. Save the contents of the received packet into node.hwinfo structure
