Detailed Description

The error functions are used to print the errors produced during a call to an RTDAL function.

When a function returns -1, the error is stored in a variable which is obtained using rtdal_error_code() and equals one of the RTDAL_ERROR_* constants.

An error message can be obtained using the functions rtdal_error_string(), rtdal_error_print() or the macro rtdal_perror().

: This part is not thread-safe.
