Detailed Description

Interface for DAC management. Each dac driver must implement all functions defined in the structure dac_interface.

Currently, dac and adc are tuned to the same frequency.

All functions return 1 if ok, or <0 if error

initialize dac. options may vary between dacs.

Sampling is started during this function call. Pointer to shared dac_cfg structure. This configuration may be changed on run-time and actions should be adopted by the dac accordingly. Pointer to write time-slot lenth. This value may be changed on run-time by the DAC. function that must be called each dac interrupt to synchronize the local processor timer (time slot).
