int hwapi_process_launch ( hwapi_process_t *  obj)

Loads a process binary into memory. The process must have been created using hwapi_process_new(). This function loads the library defined in the process attributes during the call to hwapi_process_new().

Parameters

Zero on success, -1 on error.
